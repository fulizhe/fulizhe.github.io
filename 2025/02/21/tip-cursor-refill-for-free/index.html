Cursor白嫖续杯战法

2025-02-21T06:59:40.000Z2025-03-09T12:07:14.237ZChina

概述

本教程将指导您如何使用cursor-auto-free工具实现Cursor的自动续期。整个过程需要用到 Cloudflare 和 阿里云 服务。

所需工具

  • 阿里云账号。用于购买域名。
  • Cloudflare账号。用于解析绑定好邮件转发路由。
  • GITHUB - cursor-auto-free工具。前面的准备操作都是为了正常使用它。

实现步骤

graph TD A[开始] --> B[阿里云购买域名] B --> C[将域名托管到Cloudflare] C --> D[配置Cloudflare邮件转发路由] D --> E[配置并运行该cursor-auto-free自动注册工具。期间**双手离开键盘**。] E --> G[获得新的Cursor试用账号] G --> H[结束]
  1. 整个流程其实是机器模拟cursor的注册流程，注册出新的cursor试用账户，进而实现对于cursor的持续试用。
  2. 操作大概是：
    2.1 在阿里云注册一个域名，
    2.2 然后在cf里解析绑定好邮件转发路由，
    2.3 之后使用 cursor-auto-free工具进行自动化的账号注册。
  3. 这个”cf邮件转发”就是用来接受cursor官方发来的注册验证码的。因为在cursor注册时的邮箱地址是机器自动生成的，前半部分直接就是乱码，所以我们需要将xxxx@fulizhe.xyz这样的邮件地址让cf全部转发给我们自己的邮箱里（例如xxx@gmail.com）,方便cursor-auto-free程序抓取之后自动执行下一步。

阿里云中注册域名

  • 进入域名注册页面, 选择并购买心仪的域名（建议选择便宜的域名即可）

  • 完成实名认证.

域名解析托管到cloudflare

  • 进入Cloudflare账号。

  • 域名解析托管到cloudflare。参见下面的两个链接。

    1. 详细图文手把手教你阿里云注册域名如何托管到CloudFlare DNS服务
    2. B站视频 -【完整教程】如何将阿里云域名迁移到Cloudflare DNS管理

在cf解析绑定好邮件转发路由

参考 Cloudflare Email routing rules

管理员权限下运行cursor-auto-free工具

双手离开键盘，让程序自动运行。

补充说明

获取gmail的app password

  1. 登录Google-应用专用密码
  2. 输入描述，点击“继续”
  3. 复制生成的16位密码，粘贴到.env文件的IMAP_PASS字段中。（注意Google提供的原始密码是包含空格的，你需要手动删除他们）
